# Header

Crea un header premium para una landing page SaaS moderna enfocada en marketing conversacional, automatización y creator economy.

El diseño debe sentirse:
- limpio
- moderno
- minimalista
- premium
- tecnológico
- altamente profesional

Inspirado en startups SaaS de alto nivel como:
- Stripe
- Linear
- Notion
- Manychat
- Webflow

---

# Layout General

Header horizontal transparente superpuesto y flotante sobre el hero section.

Características:
- ancho completo
- alineación perfectamente equilibrada
- spacing amplio y respirable
- altura aproximada de 80–100px
- diseño minimalista y elegante

El fondo debe ser:
- transparente
- integrado y encima visualmente del hero
- al hacer scroll, el header hace un efecto glassmorphism sutil con fondo semitransparente y blur suave
